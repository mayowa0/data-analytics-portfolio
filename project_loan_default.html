Overview
The Problem

In lending, the cost of a missed default far outweighs the cost of a rejected application. Traditional rule-based credit scoring systems often miss nuanced risk patterns — especially in emerging market borrower profiles where credit history is thin.

This project builds a binary classification model to predict loan default risk using borrower demographic, financial, and behavioral features. The goal: give a lender an early-warning signal before disbursement.
