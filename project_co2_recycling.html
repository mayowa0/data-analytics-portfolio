Context
From Research to Dashboard

This project was directly inspired by my co-authored publication on "Recycling and Reuse of Materials in Metallurgical Processes" (Harvard IJERT, 2025). The paper surfaced a critical question: how much environmental benefit is actually being captured — or lost — in Nigeria's informal recycling sector?

To answer that with data, I built an interactive Excel-based impact calculator and a Tableau dashboard that quantifies CO₂ savings, energy reduction, and material recovery rates for five key metal streams: aluminium, steel, copper, lead, and e-waste. The dashboard was designed to be usable by policymakers and sustainability managers — not just data scientists.

Process
How It Was Built
// 01
Data Collection
Gathered emission factors from IEA, World Steel, Int'l Aluminium Institute. Recycling volume data from NBS & NERC.
// 02
Excel Model
Built a dynamic calculator with emission factor tables, scenario sliders (recycling rate %), and auto-calculating CO₂ & energy outputs.
// 03
Python EDA
Cleaned and enriched data with Pandas. Ran correlation analysis between digitalization scores and recovery efficiency.
// 04
Tableau Viz
Built 4-page interactive dashboard: overview, material breakdown, zonal comparison, and trend projection to 2030.
// 05
Insight Report
Wrote a 2-page summary of findings for a non-technical audience — bridging the gap between data and policy.
